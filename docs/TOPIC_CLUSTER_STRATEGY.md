# Topic Cluster Strategy for Lunary

## What is a Topic Cluster?

A **topic cluster** is an SEO strategy where you:

1. Create a **pillar page** (main hub) covering a broad topic
2. Create **cluster pages** (supporting content) covering specific subtopics
3. Link all cluster pages back to the pillar page
4. Link related cluster pages to each other

**Example:**

- **Pillar:** "Complete Guide to Moon Magic" (`/grimoire/moon-magic-guide`)
- **Clusters:**
  - Moon Phases (`/grimoire/moon#phases`)
  - Moon Rituals (`/grimoire/moon#rituals`)
  - Moon Gardening (`/grimoire/moon#gardening`)
  - Full Moon Names (`/grimoire/moon#full-moon-names`)

## Why Topic Clusters Work

1. **Establishes Authority:** Shows Google you're an expert on the topic
2. **Improves Rankings:** Internal linking boosts all pages in the cluster
3. **Better UX:** Users can easily discover related content
4. **Higher Engagement:** More time on site, lower bounce rate

## What We Can Improve

### Option 1: Create Cross-Cluster Hub Pages

Create pages that link multiple clusters:

**Example:** `/grimoire/magical-practices-guide`

- Links to Moon Magic cluster
- Links to Candle Magic cluster
- Links to Spellcraft cluster
- Links to Divination cluster

### Option 2: Add More Internal Links

Within existing pages, add more "Related Topics" sections:

**Example:** On `/grimoire/moon#rituals`:

- Link to Candle Magic (candles used in moon rituals)
- Link to Crystals (crystals for moon magic)
- Link to Practices (spellcraft fundamentals)

### Option 3: Create Topic Hub Pages

Create landing pages for broad topics:

**Example:** `/grimoire/complete-guide-to-moon-magic`

- Overview of all moon-related content
- Links to all moon cluster pages
- Links to related practices (candle magic, crystals)
